Multi-Modal Architecture

Three specialized neural networks fused through learned gated attention
CNN (ResNet50) ——⇒ Visual features (2048 → 256D)
LSTM (BiLSTM) ——⇒ Temporal features with attention (256D)
OCR (Tesseract) ——⇒ Text features (char-level, 128D)
↓↓↓
Gated Attention Fusion —⇒ Classifier (F1: 82.2%)

Features

🔎

Real-Time Object Detection

Faster R-CNN with ResNet50-FPN backbone detects 80+ COCO object classes in every video frame with confidence scoring and bounding box visualization.

📈

Adaptive Sankey Diagrams

Object detection flow is visualized as an interactive Sankey diagram where busy segments expand and quiet ones contract, revealing the video's rhythmic structure.

🎬

Story Arc Extraction

Automatic narrative analysis identifies Opening, Rising Action, Climax, Falling Action, and Resolution phases based on detection density and object dynamics.

📊

Object Sentiment Tracking

Per-object trends (rising, falling, peaked, stable) with sparkline visualizations for both detection frequency and confidence over time.

🌐

YouTube Integration

Paste any YouTube URL — the tool downloads, trims to 60s, extracts frames, and generates a full interactive HTML analysis dashboard.

⚡

Interactive Dashboards

Self-contained HTML reports with embedded video player, clickable timeline, hover frame galleries, and synchronized Sankey-to-video navigation.

Quick Start

CLI Usage

# Install dependencies pip install -r requirements.txt # Analyze a local video python3 analyze_video.py path/to/video.mp4 # Analyze a YouTube video python3 analyze_video.py "https://youtube.com/watch?v=..." # Custom settings python3 analyze_video.py video.mp4 \ --confidence 0.5 \ --segments 12 \ --sample 3

Web Server

# Start the web UI python3 server.py # Opens http://localhost:8888 # Paste any YouTube URL and click Analyze # Custom port python3 server.py --port 9000 # Run the demo pipeline python3 pipeline.py
